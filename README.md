# Word Counter & Text Analyzer (Python)

## About

This is a simple Python project that analyzes text and provides useful statistics.

## Features

* Count total words
* Count total characters
* Count sentences
* Find most repeated word

## Tech Used

* Python

## How to Run

1. Run the Python file
2. Enter text
3. View the analysis result
